# Axute's converted addons Repository

[![Add repository on my Home Assistant][repository-badge]][repository-url]

## About
This repository contains add-ons for [Home Assistant](https://www.home-assistant.io/) that have been automatically converted from Docker images. The main goal of this project is to make any Docker container easily and quickly available as a Home Assistant add-on.

The add-ons in this repository are created using the [HAOS Add-on Converter](https://github.com/axute/haos-addon-converter).

## Installation

To use these add-ons, click the badge above or follow these steps:

1. Navigate in your Home Assistant instance to **Settings** > **Add-ons** > **Add-on Store**.
2. Click the three-dot menu (â‹®) in the top right corner and select **Repositories**.
3. Enter the following URL and click **Add**:
   `https://github.com/axute/hassio-addons-converted`
4. Close the dialog. The repository should now appear in the list.
5. You can now find the converted add-ons in the store under the section **axute's Home Assistant Add-on Repository**.

## Support
If you have issues with any of the add-ons, please create an issue in the corresponding repository or contact the maintainer.
